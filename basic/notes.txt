
TypeScript support static + dynamic typing, Type Inferencing, 8+ datatypes

JavaScript is dynamic typing


Data Types:

3. boolean
4. object (arrays are treated as object)
5. null
6. undefined
7. bigInt (value > 3^53)

All JavaScript datatypes are supported in TypeScript
9. any (make TS dynamic typing)
